# AI Photo Remix Studio
A React app built for experimenting with Cloudinary-style image remix layouts and media presentation.

## Features
- Select a demo image
- Preview a featured image layout
- Show multiple remix formats from one source image
- Prepare the UI for a Cloudinary upload flow

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- Cloudinary

## How to Run
```bash
cd frontend
npm install
npm run dev
```

## Purpose
This project was built for the Cloudinary React AI challenge as a simple image remix studio with transformation-ready previews.
