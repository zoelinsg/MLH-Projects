# Travel Mood Board Enhanced
A React travel inspiration app enhanced with Cloudinary-style media presentation and gallery layouts.

## Features
- Switch between travel destination themes
- View destination hero imagery
- Toggle between grid and highlight gallery layouts
- Show curated travel notes for each theme
- Present travel content in a polished, media-focused UI

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- Cloudinary

## How to Run
```bash
cd frontend
npm install
npm run dev
```

## Purpose
This project was built for the Cloudinary enhancement challenge to show how media presentation can improve the travel browsing experience in an existing app concept.
