# Travel Packing List Generator

A simple web application that generates customized packing lists based on your trip details.

## Description

Select your trip type, duration, weather conditions, and optional extras to instantly get an organized packing list. Perfect for travelers who want to pack smart and avoid forgetting important items.

## Features

- Multiple trip types: Beach, Mountain, City, Business
- Flexible durations: 1-3 days, 4-7 days, 8+ days
- Weather-based suggestions: Hot, Cold, Rainy
- Optional extras: Laptop, Camera, Medicine
- Organized by category: Essentials, Clothing, Electronics, Personal Care
- Responsive design for desktop and mobile
- Clear and reset functionality
- No dependencies, pure HTML, CSS, and JavaScript

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Gradients)
- Vanilla JavaScript

## How to Use

1. Open `index.html` in your web browser
2. Select your trip type, duration, weather, and optional extras
3. Click "Generate Packing List"
4. Review your organized packing list by category
5. Click "Clear" to start over
