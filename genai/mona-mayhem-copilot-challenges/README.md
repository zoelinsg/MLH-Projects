# Mona Mayhem Copilot Challenges

A React project created for the Mona Mayhem GitHub Copilot challenges.

## Completed Challenges

### Challenge 1 – The Blueprint Boss
Used GitHub Copilot Plan Mode to design a Power-Up system for Mona Mayhem before implementation. The planning process covered architecture, state flow, collision handling, temporary effect timers, and battle reset behavior. After refining the plan, Copilot scaffolded starter files and code structure for the feature, including battle UI, hooks, and game logic folders.

### Challenge 3 – The Context Crafter
Created a `.github/copilot-instructions.md` file to teach Copilot a retro-neon cyberpunk design system for Mona Mayhem. Then used those instructions to generate a Game Over screen with a game-themed layout, neon accents, and arcade-style UI language.
