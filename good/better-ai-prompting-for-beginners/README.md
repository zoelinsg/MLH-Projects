# Better AI Prompting for Beginners

A beginner-friendly, single-page educational website that teaches how to write clearer prompts for AI tools, craft better specifications, and safely upload images when using AI services.

Why this project is useful
- Helps newcomers quickly learn a simple, repeatable prompt formula.
- Shows practical tips for safer image use and specification writing.
- Provides a small interactive example generator so learners can practice.

Features
- Explanations of what makes a good AI prompt (goal, context, constraints, output format).
- A simple prompt formula: **Goal + Context + Constraints + Output Format**.
- Guidance on writing better specifications (audience, features, tech, style, avoidances).
- Safe image upload tips and privacy reminders.
- Common beginner mistakes and how to fix them.
- A small interactive prompt example generator (task, audience, output format).

Technologies used
- HTML, CSS, JavaScript
- No build step or backend — static site

How to run locally
1. Clone or download this repository and open the project folder.
2. Option A — Quick static server (Python 3):

```bash
# from the project folder
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

3. Option B — Use VS Code and the Live Server extension: open the folder, right-click `index.html` and choose "Open with Live Server".

What learners will gain
- A practical, repeatable prompt-writing formula they can use with tools like ChatGPT or Copilot.
- Tips for writing clear specifications that reduce ambiguity.
- Awareness of privacy and safety when uploading images to AI tools.
- Hands-on practice via the interactive prompt generator.

Connections to digital literacy, safer AI use, and Hacking for Good
- Digital literacy: Teaches learners how to communicate effectively with AI systems and evaluate outputs critically.
- Safer AI use: Includes concrete steps to avoid sharing sensitive personal information and to verify AI-generated content before trusting it.
- Hacking for Good: Encourages responsible, thoughtful use of AI to solve problems — starting with clear problem statements and constraints.
