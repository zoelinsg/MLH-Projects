# Kindness Filter for Social Good

Kindness Filter for Social Good is a beginner-friendly coded concept project that demonstrates how a social media-style filter can be used to promote kindness, empathy, and safer online communities.

## Features
- Social media story-style filter preview
- Multiple kindness-themed filter messages
- Social good explanation panel
- Friendly community awareness content
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to show how digital filters and visual messaging can support positive online behavior, anti-bullying awareness, and stronger community care.

## How to Run
Open `index.html` in your browser.
