# Budget Buddy

Budget Buddy is a beginner-friendly financial literacy web app that helps users practice budgeting, understand spending categories, and build better saving habits.

## Features
- Enter monthly income
- Track expenses by category
- Calculate total expenses
- View remaining balance
- See savings rate
- Get simple financial tips

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open `index.html` in a browser.

## Purpose
This project was created to help beginners learn basic personal finance concepts such as budgeting, spending awareness, and saving.
