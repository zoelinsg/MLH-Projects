# Mona Mayhem Arena Lab

A beginner-friendly retro-neon cyberpunk arcade web app prototype built with HTML, CSS, and JavaScript.

## Overview

Mona Mayhem Arena Lab is a fictional battle arena interface designed as a learning project. It showcases a neon arcade homepage, battle dashboard UI, and a dynamic Match History feature while exploring GitHub Copilot workflows and feature planning.

## Features

- Retro-neon cyberpunk arcade homepage
- Battle arena dashboard UI
- Player stats panel
- Battle status panel
- Game Over screen
- Match History feature with win/loss styling
- localStorage support for saved match history
- Clear history action
- Power-Up system blueprint and starter files

## Copilot Workflow Highlights

- `.github/copilot-instructions.md` provides custom design and project context
- `POWERUP_PLAN.md` documents the power-up system architecture
- Copilot assisted the app UI expansion and feature refinements
- The Match History feature was added and refined with Copilot guidance

## File Structure

- `index.html` — main application structure and modals
- `style.css` — retro-neon arcade styling
- `script.js` — UI behavior, modals, and Match History logic
- `player-state.js` — player data helper file
- `powerup-system.js` — power-up system scaffold
- `game-loop.js` — arena loop and interaction logic
- `.github/copilot-instructions.md` — Copilot design instructions
- `POWERUP_PLAN.md` — power-up system planning document

## Run Locally

1. Open the project folder in your browser or code editor.
2. Open `index.html` in a browser.
3. Interact with the buttons to view the arena, Game Over screen, and Match History.
