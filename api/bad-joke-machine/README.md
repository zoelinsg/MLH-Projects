# Bad Joke Machine

Bad Joke Machine is a simple useless hack built with React and Vite.  
It uses JokeAPI to generate random bad jokes with the click of a button.

## Features

- Generate random bad jokes
- Fetch joke data from JokeAPI
- Clean and simple user interface
- Built as a fun API-powered web app

## Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS
- JokeAPI
- Fetch API

## API Used

[JokeAPI](https://v2.jokeapi.dev/)

## Getting Started

1. Clone the repository
2. Navigate to the project folder
3. Install dependencies
```bash
npm install
```
4. Start the development server
```bash
npm run dev
```
## Project Idea
This project was created for a hackathon challenge to build a useless invention that uses an API.
The result is a playful web app that generates random terrible jokes for entertainment.
