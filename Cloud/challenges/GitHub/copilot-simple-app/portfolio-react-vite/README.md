# Copilot Simple App — Portfolio (React + Vite)

A simple single-page portfolio website built with **React + Vite**, created and iterated using **GitHub Copilot** as an AI coding assistant.

## What this project includes
- Single-page sections: **Hero / Skills / Projects / Experience / Contact**
- **Navbar smooth scroll** to each section
- **Responsive layout** for mobile and desktop
- (Optional) Production build verified with `npm run build`

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Getting Started
From this folder:

```bash
npm install
npm run dev
