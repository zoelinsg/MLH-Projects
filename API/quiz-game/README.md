# Trivia Quiz Game

A simple quiz game built with React and Vite that fetches questions from the Open Trivia DB API.

## Features

- Start a quiz with 5 multiple-choice questions
- Fetch quiz data from a public API
- Show correct and incorrect answers with instant feedback
- Track player score
- Restart the game and play again

## Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS
- Open Trivia DB API
- Fetch API

## How It Works

This project uses the Open Trivia DB API to load quiz questions dynamically. The player answers multiple-choice questions, gets immediate visual feedback, and receives a final score after completing the quiz.

## Getting Started

1. Clone the repository
2. Go to the project folder
3. Install dependencies

```bash
npm install
```
4. Start the development server
```bash
npm run 
```
