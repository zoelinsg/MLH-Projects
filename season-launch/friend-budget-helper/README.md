# Friend Budget Helper

## Core Goal

Build something useful for a friend for MLH Global Hack Week: Season Launch.

## Project Overview

**Friend Budget Helper** is a simple split-bill calculator for friends.

It helps calculate how much each person should pay when sharing the cost of a meal, cafe order, trip expense, or group activity.

## Why I Built This

I built this project for a friend who often goes out with classmates and needs a quick way to split shared expenses.

Instead of calculating everything manually, this tool allows my friend to enter the total amount, number of people, service fee, and discount. The app then calculates the amount each person should pay.

## Features

- Enter total bill amount
- Enter number of people
- Add optional service fee percentage
- Add optional discount amount
- Calculate how much each person should pay
- Show a clear payment summary
- Validate invalid inputs

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub

## How to Run

Open `index.html` in a browser.

No installation is required.
