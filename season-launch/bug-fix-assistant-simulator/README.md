# Bug Fix Assistant Simulator

## Project Overview

Bug Fix Assistant Simulator is a beginner-friendly JavaScript command line game built with Node.js and the readline module. The player is presented with common programming bug scenarios, chooses the best debugging action, receives feedback, and finishes with a final score.

This project was created for MLH Global Hack Week as a simple way to practice debugging skills in a fun and interactive format.

## Challenge Goals

This project supports the following goals:

1. Build a small complete application with JavaScript
2. Use GitHub Copilot to help create and improve the project
3. Practice problem-solving and beginner-friendly debugging concepts
4. Create a polished project that is easy to run and understand

## How GitHub Copilot Helped

GitHub Copilot assisted with:

- generating the initial project structure
- suggesting game logic and scenario ideas
- helping organize the code into an array of reusable scenarios
- improving the README and overall project clarity

Copilot helped speed up development, while I still reviewed the code, tested the game, and made sure the experience was beginner-friendly.

## Features

- Multiple bug scenarios with realistic error messages
- Three possible debugging actions for each scenario
- Immediate feedback after each choice
- Score tracking throughout the game
- A final summary at the end
- Simple and clear command line gameplay

## Tech Stack

- JavaScript
- Node.js
- Node.js readline module
- GitHub Copilot

## How to Run

From the project folder, run:

```bash
npm start
```

The game will guide you through each bug scenario and ask you to choose the best debugging action.
