# Cursor Tip Calculator

A small tip calculator app built with Cursor for the MLH Cursor challenges.

## Features

- Enter a bill amount
- Choose a preset tip percentage
- Enter a custom tip percentage
- Split the total by number of people
- Show tip amount, total amount, and per-person amount
- Validate invalid split values, such as 0 people

## Built With

- HTML
- CSS
- JavaScript
- Cursor

## Run Locally

Open `index.html` in your browser.

## Cursor Challenges

This project was used for:

- Challenge 3: Build a Mini App with Cursor
- Challenge 4: Build a Feature on an Existing Project
- Challenge 5: Squash a Bug with Cursor

Cursor helped scaffold the app, add the split bill feature, and fix the zero-people split bug by showing a helpful validation message instead of an invalid result.
