# Solana Keypair and Devnet SOL

A small Solana exercise that generates a new keypair programmatically with `@solana/kit`.

## Run

```bash
npm install
node create-wallet.mjs
```

The generated public address can be funded with test SOL through the Solana Devnet faucet.

## Built With
- JavaScript
- Node.js
- Solana Devnet
- @solana/kit

Devnet only. No real SOL is used.
