# Cursor Challenge 2

A small Python example used to practice Cursor AI Chat.

## What I Did

I created a simple Python function that reverses a string, then used Cursor AI Chat to explain what the function does in plain English.

## Code Example

```python
def reverse_string(text):
    return text[::-1]

print(reverse_string("Cursor"))
```
