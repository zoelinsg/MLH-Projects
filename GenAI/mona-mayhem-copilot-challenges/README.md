# Mona Mayhem Copilot Challenges
A React demo project for the Mona Mayhem GitHub Copilot challenges.

## Focus of this version
This version focuses on **Challenge 3: The Context Crafter**.

I created a `.github/copilot-instructions.md` file to teach GitHub Copilot the retro-neon cyberpunk design system for Mona Mayhem. Then I used those instructions to generate a **Game Over** screen with a game-themed layout, neon accents, arcade-style labels, and bold action buttons.

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- GitHub Copilot

## How to Run
```bash
cd frontend
npm install
npm run dev
```

## Files
- .github/copilot-instructions.md
- frontend/src/App.jsx
- frontend/src/App.css

## Purpose
This project demonstrates how custom workspace instructions can guide Copilot to generate UI that matches a specific design system and game aesthetic.
